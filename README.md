# SAR Research Notes

A personal research-note repository for SAR signal processing topics, including derivations, surveys, reading notes, and teaching-style notebooks.

## Scope

This repository is a topic-based knowledge workspace rather than a single implementation project.
Each topic lives in its own folder and may contain:

- derivations
- survey notes
- figures
- notebooks
- paper references

## Topics

- [TOPS](TOPS/README.md)  
  Notes on TOPS SAR azimuth processing, including folding, UFR, deramping, reramping, and azimuth compression.

- [DPCA](DPCA/)  
  Notes and survey material on displaced phase center antenna methods.

## Repository Structure

```text
sar_research_note/
├── README.md

```
